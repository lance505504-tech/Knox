PIN Reset Emails

Knox sends PIN reset codes by email using EmailJS — a free service that lets websites send email without a server. Setup takes about 5 minutes and is free for up to 200 emails/month.

1 Create a free EmailJS account emailjs.com
→ Go to emailjs.com and sign up free

The free plan gives you 200 emails/month — more than enough for PIN resets. No credit card required.

2 Add an email service Dashboard → Email Services
→ Open EmailJS Dashboard

Click Email Services → Add New Service. Choose Gmail, Outlook, or any provider you use. Follow the prompts to connect your email account.

Once connected, copy your Service ID — it looks like service_xxxxxxx

3 Create the Knox reset template Dashboard → Email Templates
→ Go to Email Templates

Click Create New Template. Then:

1. Set Subject to: Knox PIN reset

2. Set To Email (right panel) to: {{to_email}}

3. Click Edit Content → HTML view and paste this exactly:

<div style="font-family:system-ui,sans-serif;font-size:14px;color:#2c3e50;max-width:480px;">
  <p>A PIN reset has been requested for your Knox vault.</p>
  <div style="margin:24px 0;padding:24px;background:#f0f7ff;border-radius:8px;text-align:center;">
    <div style="font-size:32px;margin-bottom:12px;">🔐</div>
    <div style="font-size:13px;color:#888;margin-bottom:8px;">Your reset code</div>
    <div style="font-size:36px;font-weight:bold;letter-spacing:10px;color:#2c3e50;">{{code}}</div>
    <div style="font-size:12px;color:#aaa;margin-top:12px;">Expires in 15 minutes</div>
  </div>
  <p style="font-size:12px;color:#aaa;">If you didn't request this, ignore this email. Your PIN has not changed.</p>
</div>

4. Save the template and copy the Template ID — it looks like template_xxxxxxx

4 Get your Public Key and paste everything below Account → API Keys
→ Go to Account → API Keys

Copy your Public Key from the API Keys section. Then paste all three values into the fields below and you are done.
